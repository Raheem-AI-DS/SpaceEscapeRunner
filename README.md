## Overview

SpaceEscape Runner is an arcade-style spaceship survival game where the player navigates through obstacles and tries to achieve the highest score. The project follows a clean and modular architecture for better scalability and maintenance.

---

## Features

- Smooth spaceship movement system  
- Touch-based controls  
- Modular and reusable components  
- Optimized performance for mobile devices  
- Expo-based cross-platform support  
- Clean and maintainable code structure  

---

## Project Structure

```
SpaceEscapeRunner/
│
├── app/                # Application screens and routing
├── assets/            # Images, icons, sounds
├── src/               # Core game logic
│   ├── components/    # Reusable UI components
│   ├── logic/         # Game mechanics
│   ├── utils/         # Helper functions
│
├── constants/         # App constants
├── hooks/             # Custom React hooks
├── app.json           # Expo configuration
├── package.json       # Dependencies
└── babel.config.js    # Babel setup
```

## Installation

### Clone Repository
```bash
git clone https://github.com/your-username/SpaceEscapeRunner.git
```

### Navigate to Project Folder
```bash
cd SpaceEscapeRunner
```

### Install Dependencies
```bash
npm install
```

---

## Running the Project

### Start Development Server
```bash
npx expo start
```

### Run on Device or Emulator
- Install Expo Go on mobile device
- Scan QR code from terminal
- Or run using Android/iOS emulator
